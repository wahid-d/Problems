# Remove Special Characters

Create a function that takes a string, removes all **special** characters (e.g. **`!, @, #, $, %, ^, &, \, *, (, )`**  ) and returns the new string. The only non-alphanumeric characters allowed are dashes **`-`**, underscores **`_`** and spaces.

### Examples
```cs
RemoveSpecialCharacters("The quick brown fox!") â "The quick brown fox"

RemoveSpecialCharacters("%fd76$fd(-)6GvKlO.") â "fd76fd-6GvKlO"

RemoveSpecialCharacters("D0n$c sed 0di0 du1") â "D0nc sed 0di0 du1"
```
### Notes
* You can solve this challenge with `RegEx`, but it's intended to be solved with logic.
